# Machines Listening to Themselves

Machines Listening to Themselves is a distributed audiovisual installation exploring machine listening, industrial data, and generative audiovisual systems.

The project investigates how industrial infrastructures produce vast streams of invisible information: vibrations, airflow, mechanical rhythms, and energy flows.

Through field recordings and industrial sensors, these signals are captured and analyzed using custom software and AI-assisted pattern recognition.

The system transforms these signals into evolving sound synthesis and generative visual structures.

Instead of representing machines as passive tools of production, the project reframes them as sensing agents capable of perceiving and expressing their own operational states.

## System Architecture

Industrial machines generate acoustic and mechanical signals.

Sensors capture these signals.

AI-assisted analysis identifies patterns and anomalies.

The data is translated into sound synthesis and generative visual systems.

The result is a continuously evolving audiovisual ecosystem.

The installation is designed as an immersive audiovisual environment consisting of:

- large-scale projection
- multi-channel audio system
- real-time generative visual engine
- industrial field recordings and sensor data
